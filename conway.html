Conway's Game of Life

Conway's Game of Life is a pattern generating game. That patterns exist on a grid, where cells in the grid are either 'alive' or 'dead'. Based on the current state of the grid, the next generation will be generated based on certain rules about the cells and their neighbors in the current state of the grid.

    Here are the rules (from Wikipedia):
  1. Any live cell with fewer than two live neighbours dies, as if by underpopulation.
  2. Any live cell with two or three live neighbours lives on to the next generation.
  3. Any live cell with more than three live neighbours dies, as if by overpopulation.
  4. Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

I implemented the game using the JavaScript and React.

The app will automatically loaded with a randomly seeded initial state and start generating successive states until stable patterns or oscillating patterns emerge.

You can use the buttons to start and the stop the generation, initialize the grid to a random seed, and clear the grid (all cells are 'dead'). You can also click on individual cells to manually change the cells from their previous state!
